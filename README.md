# Space Invaders

A Space Invaders clone using HTML canvas.

Currently has:
- a blank canvas to draw in
- an alien swarm moving about the canvas
- a player moving with arrow keys

To add:
- bullets

## To run

0. Clone repository
0. Run `npm install`
0. Run `npm start` - the app should be live at `http://localhost:3000/`
